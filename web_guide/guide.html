ThinkPad T490s Installation Protocol

Bypassing Error 0x8007000D and the Invisible SSD Paradox

Phase 1: USB Architecture

Standard USB tools fail with modern Windows ISOs. We use a dual-partition layout.

  • Partition 1 (FAT32): 1GB for UEFI/Boot files.
  • Partition 2 (NTFS): Remainder for the large installation image.

Phase 2: The Invisible SSD

If the drive list is empty, don't panic. The T490s needs Intel RST v18.7 drivers.

1. Use SetupRST.exe -extractdrivers to get the raw .inf files.

2. Put them in a /DRIVERS folder on your USB.

Phase 3: God-Mode Manual Deploy

When the GUI fails, use DISM directly from the Shift+F10 console.

diskpart
select disk 0
clean
convert gpt
...
dism /apply-image /imagefile:D:\sources\install.wim /index:1 /applydir:W:\
